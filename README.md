**Discord.js v14**\n- **Socket.io** للتحديثات الفعلية\n- **بدون قاعدة بيانات** (JSON Storage)\n\n### Frontend\n- **Next.js 14**\n- **React 18**\n- **TailwindCSS**\n- **Dark Mode + Glassmorphism**\n\n## 📱 التثبيت على Termux\n\n### المتطلبات\n```bash\npkg update && pkg upgrade -y\npkg install nodejs git python-pip\n```\n\n### خطوات التثبيت\n\n```bash\n# استنساخ المشروع\ngit clone https://github.com/ggvjvhhvjgfc-cyber/x10think-discord-dashboard.git\ncd x10think-discord-dashboard\n\n# تثبيت Backend\ncd backend\nnpm install --legacy-peer-deps\n\n# تثبيت Frontend\ncd ../frontend\nnpm install --legacy-peer-deps\n\n# العودة للجذر\ncd ..\n```\n\n### التكوين\n\n**نسخ ملف البيئة:**\n```bash\ncp .env.example .env\n```\n\n**تعديل `.env` بـ Termux:**\n```bash\nnano .env\n# أو\nvim .env\n```\n\n**إضافة بيانات Discord:**\n```\nDISCORD_CLIENT_ID=your_client_id\nDISCORD_CLIENT_SECRET=your_client_secret\nDISCORD_TOKEN=your_bot_token\n```\n\n### التشغيل\n\n**Terminal 1 - Backend:**\n```bash\ncd backend\nnpm start\n```\n\n**Terminal 2 - Frontend:**\n```bash\ncd frontend\nnpm run dev\n```\n\n### الوصول\n- **Dashboard:** http://localhost:3000\n- **API:** http://localhost:5000\n- **Health Check:** http://localhost:5000/health\n\n## 🔑 إعدادات Discord Bot\n\n1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)\n2. اضغط **New Application**\n3. اذهب إلى **Bot** → **Add Bot**\n4. انسخ **Token** → ضعه في `.env` كـ `DISCORD_TOKEN`\n5. اذهب إلى **OAuth2 → General**\n6. انسخ **Client ID** → ضعه في `.env`\n7. انسخ **Client Secret** → ضعه في `.env`\n8. اضبط **Redirect URI:** `http://localhost:3000/api/auth/callback`\n9. اذهب إلى **Bot → Scopes** واختر:\n   - `bot`\n   - `identify`\n   - `email`\n   - `guilds`\n10. اختر **Permissions:**\n    - Administrator (أو الأمانات المحددة)\n\n## 📁 هيكل المشروع\n\n```\nx10think-discord-dashboard/\n├── backend/\n│   ├── src/\n│   │   ├── index.js              # نقطة البداية\n│
